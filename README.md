🐱 MeowMart Web

MeowMart Web adalah sistem manajemen petshop berbasis web yang dikembangkan untuk membantu operasional toko seperti POS, manajemen produk, dan transfer stok antar cabang.

🚀 Tech Stack

Vue 3
TypeScript
Vite
REST API Integration

📌 Fitur Utama

🛒 Point of Sale (POS)
📦 Manajemen Produk
🔄 Transfer Stok Antar Cabang
👤 Registrasi Customer oleh Admin
📑 Laporan Sederhana
📊 Dashboard Ringkasan Penjualan
